# BB369TECH China Website Preview Module v1.0.0

中文优先、纯前端、低数据的企业官网视觉预览模块。

## 部署位置

将整个 `preview/` 文件夹复制到 `china.bb369tech.com` 对应仓库根目录。

最终访问路径：

`https://china.bb369tech.com/preview/`

## 数据边界

- 不使用账号
- 不使用数据库
- 不上传用户输入
- 不使用 Cookie 或本地持久化
- 不调用生成式 AI
- 不收集联系人、手机号或证件信息

## 入口链接

在中国官网首页添加：

```html
<a href="preview/">免费企业官网预览</a>
```
